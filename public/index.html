POST  api/properties/{property_id}/reviews

Creates a review for the specified property. A body must be passed containing a valid guest_id and rating, the comment is optional.

{
    guest_id: 1,
    rating: 5,
    comment: "Lovely getaway for the family, very clean and tidy!",
}

Returns with Status 201 on success.

DELETE  api/reviews/{review_id}

Removes a review. On success returns Status Code 204.
